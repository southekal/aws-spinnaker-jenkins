Setup Jenkins for Continuous Integration

  • In order to create a simple pipeline service that Spinnaker can execute, lets setup Continuous Integration using Jenkins
  • Set up Jenkins as an AWS EC2 service. Refer to https://d1.awsstatic.com/Projects/P5505030/aws-project_Jenkins-build-server.pdf that goes over the task of creating a Jenkins server that allows SSH and outside world access through port 8080
  • Create a new Jenkins job of type Multibranch pipeline that looks at the appropriate git repository to build.
  • Next, we will trigger Spinnaker pipelines with Jenkins

Things to look out for

  • In the Global Tool Configuration section of Jenkins, ensure that you have installed project requirements such as: maven, java, docker etc
  • If you run into issues with Jenkins being unable to run docker then fix user permissions with sudo usermod -a -G docker jenkins and restart the jenkins service with sudo service jenkins restart on the EC2 server.
