AWS IAM Creation

  • Create a new user utilizing AWS IAM policy. Attach an AdministratorAccess policy for the sake of getting started. Refer aws documentation on how to create an IAM policy utilzing AWS console.
  • Install AWS CLI. Refer aws documentation to install the cli based on your system.
  • Configure your AWS CLI to use the user credentials generated via the IAM policy. Refer aws documentation for generating the ~/.aws/credentials file
