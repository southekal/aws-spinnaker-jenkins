Triggering pipelines with Jenkins

  • First, lets connect Jenkins to Spinnaker. Refer to https://www.spinnaker.io/setup/ci/jenkins/ and redeploy spinnaker with hal deploy apply
  • Once, Jenkins is connected to Spinnaker we can go ahead and add a Jenkins trigger as described here: https://www.spinnaker.io/guides/user/pipeline/triggers/jenkins/

Congrats! you have now succesfully launched a Spinnaker instance that triggers a pipleline with Jenkins CI
