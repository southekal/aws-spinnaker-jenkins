Install kubectl

  • kubectl is a command line tool for controlling Kubernetes clusters. kubectl looks for a file named config by default in the ~/.kube directory. In this tutorial we will use the default config found in ~/.kubectl but you can specify other kubeconfig with the --kubeconfig parameter.
  • Install kubectl. Refer kubectl installation documentation to install based on your system.
  • Verify that kubectl is available in your $PATH via kubectl version --client
