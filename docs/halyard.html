Install halyard

  • halyard is a command line tool that manages the lifecycle of Spinnaker deployment including writing & validating your deployment’s configuration, deploying each of Spinnaker’s microservices, and updating the deployment.
  • Install halyard. Refer halyard installation documentation to install based on your system. In this flow, we will be installing halyard locally on our machine.
  • Verify that halyard is available in your $PATH via hal -v

Common issues

  • on a linux system when locally installing halyard, you might run into a /usr/lib/sysusers.d/halyard.conf: No such file or directory issue. Manually creating a directory mkdir /usr/lib/sysusers.d allows the installation process to go through successfully.
